Class: Delegate

Defined in: lib/cocoa-delegate.coffee

Overview

Note: It is important to call init() at the end to register and properly instantiate the delegate class.

Delegate class for all UI elements to create a Cocoa delegate that will implement the callbacks and hooks.

Examples:

How to create a delegate 

class SomeControl
  constructor: (tag, parent, title) ->
    @nscontrol = $NS...("alloc")("init...", ...)
    @delgate = new Delegate("CocoacClassName")

  init: ->
    delegate.init()
    @nscontrol("setTarget", @delegate.init())  # or "setAction" depending on which Cocoa control is used

# (Delegate) addMethod(event, signature, callback)

Add an instance method to the delegate class

Remember to call init() after last addMethod call

Examples:

delegate = new Delegate("Example")
delegate.addMethod("onClick", "v@:@", (-> console.log "called"))

Parameters:

  • event ( String ) — name of the method
  • signature ( String ) — type encoded signature (see https://developer.apple.com/library/mac/documentation/Cocoa/Conceptual/ObjCRuntimeGuide/Articles/ocrtTypeEncodings.html)
  • callback ( Function ) — the function to be called when the event is fired

Returns:

  • ( Delegate ) — the node delegate instance to allow chaining

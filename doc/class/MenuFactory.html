# (MenuItem) createMenuItem(names, item, menus)

Create a MenuItem instance for a given menuconfig item

Parameters:

  • names ( Array<String> ) — List of menu item entry names in one hierachy of sub menus
  • item ( Object ) — Object of form {title:"...", action:..., key:"...", modifiers:[...]} describing the menu item behaviour
  • menus ( Object ) — The resulting tree of submenus down from the root menu entry

Returns:

  • ( MenuItem ) — The MenuItem instance created in this step

Create a sub menu in order to allow menu hierarchies

Parameters:

  • names ( Array<String> ) — List of menu item entry names in one hierachy of sub menus
  • menu ( Object ) — One single menu hierarchy (array) in a menuconfig object
  • menus ( Object ) — The resulting tree of submenus down from the root menu entry

Returns:

  • ( MenuItem ) — The sub menu item created

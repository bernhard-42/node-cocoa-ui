Class: ScrollView

Defined in: lib/cocoa-textview.coffee
Inherits: UiControl

Overview

The ScrollView class creates a Cocoa scroll view and wraps them into a coffeescript object

Examples:

textView = new TextView("textview", window)
scrollview = new ScrollView("scrollview", window, textview)
textview.appendText "Some long text"

Constructor Details

# (void) constructor(tag, parent, textView, hscroll = true, vscroll = true)

Create a new ScrollView for a TextView. The Cocoa NSScrollView instance can be accessed via property "nscontrol"

Parameters:

  • tag ( String ) — Name or tag of the Scrollview instance (each control of a one type must have a different tag)
  • parent ( Window ) — The Window instance this button is connect with
  • textview ( TextView ) — The textview to embed into the ScrollView
  • hscroll ( Boolean ) — Allow horizontal scrolling
  • vscroll ( Boolean ) — Allow vertical scrolling

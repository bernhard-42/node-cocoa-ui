Class: Slider

Defined in: lib/cocoa-slider.coffee
Inherits: UiButton

Overview

Note: It is important to call init() at the end to register and properly instantiate the delegate class.

The Slider class creates a Cocoa slider and wraps them into a coffeescript object

Examples:

slider = window.addSlider("slider", 0, 10)
.numTicks 11
.snapToTicks true
.setValue 5
.onMoved -> console.log(slider.getValue())
.init()

Constructor Details

# (void) constructor(tag, parent, minValue = 0, maxValue = 1, circular = false)

Create a new slider. The Cocoa NSSlider instance can be accessed via property "nscontrol"

Parameters:

  • tag ( String ) — Tag of the Slider instance (each control of a one type must have a different tag)
  • parent ( Window ) — The Window instance this button is connect with
  • minValue ( Integer ) — Minimum value of slider
  • maxValue ( Integer ) — Maximum value of slider
  • circular ( Boolean ) — Circular (true) or linear slider (false)

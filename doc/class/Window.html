Class: Window

Defined in: lib/cocoa-window.coffee

Overview

Note: It is important to call init() at the end to register and properly instantiate the delegate class.

Window class creates a Cocoa window and wraps it into a coffeescript object

Examples:

window = new Window(app, "mainWindow", title, ["Titled", "Resizable", "Miniaturizable", "Closable"])
.cascadeTopLeftFromPoint 20, 20
.onResize ->   debug "resized"
.onMinimize -> debug "minimize"
.onMaximize -> debug "maximize"
.onClose ->    debug "close"
.init()

# (void) constructor(app, tag, title, styles, width = 0, height = 0)

Create a new Window. The Cocoa NSWindiw instance can be accessed via property "nswindow"

@param width [] Width of window default=0. Usually not used, due to Autolayout support @param height [] Height of window default=0. Usually not used, due to Autolayout support

Parameters:

  • app ( App ) — The App instance containing the window
  • tag ( String ) — Tag of window (each window must have a different tag)
  • title ( String ) — Window title
  • styles ( Array<String> ) — Style Strings can be: "Titled", "Resizable", "Miniaturizable", "Closable"

# (void) openDialog(callback, options = {})

Open the system file open dialog as a modal dialog of the window

return [NSOpenPanel] the Cocoa dialog instance

Examples:

options =
  folder:"/Users/bernhard/Development"
  multipleSelection: true
  chooseFolders: true
  chooseFiles: true
  hiddenFiles: true
  title: "Test open"
  prompt: "Get it"
  message: "Select multiple files as you like"

window.openDialog((selection) ->
  console.log selection
, options)

Parameters:

  • callback ( function ) — The function to be called after file(s) have been selected or dialog has been cancelled

Options Hash: (options):

  • folder ( String ) — Starting folder
  • multipleSelection ( Boolean ) — Allow to select more than one file
  • chooseFolders ( Boolean ) — Allow to select folders (instead of opening them)
  • chooseFiles ( Boolean ) — Allow to select files
  • hiddenFiles ( Boolean ) — Show hidden files/folders
  • title ( String ) — Title of the dialog window
  • prompt ( String ) — Prompt shown in the open File Dialog
  • message ( String ) — Message shown in the open File Dialog

# (ProgressIndicator) addProgressIndicator(tag, minValue, maxValue)

Convenience method to add a ProgressIndicator to a window If minValue or maxValue or both are omitted, an indeterminate spinning "wheel" will be shown, else a linear progress indicator

Examples:

# indeterminate

# determinate

Parameters:

  • tag ( String ) — Name or tag of the Slider instance
  • minValue ( Intger ) — Minimum value of the slider (optional)
  • maxValue ( Integer ) — Maximum value of the slider (optional)

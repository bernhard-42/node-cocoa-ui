~ (NSOpenPanel) openDialog(nswindow, callback, options)

Open the system file open dialog as a modal dialog of the window

Examples:

options =
  folder:"/Users/bernhard/Development"
  multipleSelection: true
  chooseFolders: true
  chooseFiles: true
  hiddenFiles: true
  title: "Test open"
  prompt: "Get it"
  message: "Select multiple files as you like"

nonModalDialog = openDialog(null, (selection) ->
  console.log selection
, options)

modalDialog = openDialog(nswindow, (selection) ->
  console.log selection
, options)

Parameters:

  • nswindow ( NSWindow ) — if null a non modal dialog will be opened, else a model dialog of the given NSWindow instance
  • callback ( function ) — The function to be called after file(s) have been selected or dialog has been cancelled

Options Hash: (options):

  • folder ( String ) — Starting folder
  • multipleSelection ( Boolean ) — Allow to select more than one file
  • chooseFolders ( Boolean ) — Allow to select folders (instead of opening them)
  • chooseFiles ( Boolean ) — Allow to select files
  • hiddenFiles ( Boolean ) — Show hidden files/folders
  • title ( String ) — Title of the dialog window
  • prompt ( String ) — Prompt shown in the open File Dialog
  • message ( String ) — Message shown in the open File Dialog

Returns:

  • ( NSOpenPanel ) — The Cocoa file dialog instance

Open the system file save dialog as a modal dialog of the window

options =

  chooseFiles: true
  hiddenFiles: true
  title: "Test save"
  prompt: "Save it"

  console.log selection
, options)

  console.log selection
, options)

Parameters:

  • nswindow ( NSWindow ) — if null a non modal dialog will be opened, else a model dialog of the given NSWindow instance
  • callback ( function ) — The function to be called after a filename has been selected or dialog has been cancelled

Options Hash: (options):

  • folder ( String ) — Starting folder

  • hiddenFiles ( Boolean ) — Show hidden files/folders
  • title ( String ) — Title of the dialog window
  • prompt ( String ) — Prompt shown in the save File Dialog
  • message ( String ) — Message shown in the save File Dialog

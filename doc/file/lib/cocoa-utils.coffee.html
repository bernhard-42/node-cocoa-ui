~ (NSMutableAttributedString) createAttributedString(text, options)

Create an NSAttributedString from a text and attribut options

createAttributedString("Some attributed text", options)

Examples:

options =
  font: "Calibri"
  size: 24
  color: rgbArray
  face: "ubi"

Parameters:

  • text ( String ) — The text to be converted

Options Hash: (options):

  • font ( String ) — A valid OS X font name
  • size ( Integer ) — Font size of the text to be inserted
  • face ( String ) — String of characters indicating font face: i=italics, b=bold
  • color ( Array<Float> ) — A 3-dimensionla array with values between 0 and 1 encoding an RGB color

Returns:

  • ( NSMutableAttributedString ) — The converted Cocoa attributed String
